Bracketup

Author: Philip Dorrell

Bracketup is a generic textual markup language which allows the definition of simple application-specific markup languages which can be used to embed content in web pages (via <script> elements), such that the content is compiled in the browser into HTML (in accordance with the requirements of the specific application).

Examples

The following examples demonstrate Correspondence-Bracketup, which is a markup language based on Bracketup. Correspondence-Bracketup compiles to the HTML format required by Correspondence.
